Instructions

Divide up grocery lists like a boss. Enter the price of each item either as dollars with a decimal point or as cents without

100 and 1.00 both enter 1.00

Multiply with @ or *

4@3.00, 4*300, 2 @ -1.30, etc.

Precede a series of prices with a string of letters to denote ownership. Case and order don't matter!

MT indicates that all following prices are split between M and T.

Enter ! to remove items from the current owners in reverse order (alternatively enter negative items).

Enter $ to see a summary of data entered so far.

Start with the first owner string

Instructions

Divide up grocery lists like a boss. First click, the big + to create a new shopping list. Type some letters to describe who paid for this list.

If the whole list was paid for by Jebediah, you might enter J. If the cost of the list was split between Jebediah and Roland, you might enter JR.

The order of the letters doesn't matter, nor does upper- vs. lower-case.

Next enter who the first item was bought for, using a similar string of letters.

If Hank and Linda are splitting a $5.49 gallon of milk, you would first enter HL.

Next you can start entering prices.

Enter dollars with a decimal point or cents without. 100, 1.00, and 1. all enter $1.00.

Multiply with @ or *

4@3.00, 4*300, 2 @ -1.30, etc.

Click an entry in a shopping list to remove it. Click shopping lists to switch which list the next price is added to. Click "Total" to see the results of all current lists.
